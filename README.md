# Hotel Almirante - Plantillas WitBooking

Proyecto de personalización de plantillas para el motor de reservas y emails del Hotel Almirante usando WitBooking.

## Sistema de plantillas

WitBooking usa **Mustache** como lenguaje de plantillas. Las plantillas son HTML con estilos inline y un máximo de 64KB.

## Tipos de plantillas

- **Confirmación de reserva** - Email enviado al confirmar una reserva
- **Pre-estancia** - Email enviado antes de la llegada
- **Post-estancia** - Email enviado después de la estancia
- **Motor de reservas** - Personalización visual del motor web

## Herramientas útiles

- Validación Mustache: https://pismute.github.io/Try-Dashbars/
- Test de emails: https://putsmail.com/tests/new
- Minificar HTML: https://codebeautify.org/minify-html
